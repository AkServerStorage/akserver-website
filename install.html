Quick Overview

This page provides a practical, step-by-step installation guide for AkServer on Windows. Follow each step carefully. Troubleshooting tips are at the bottom of this page.

Prerequisites

  • Windows 10 / 11 (x64) or compatible PC
  • Administrator privileges to install software
  • Local network connectivity (Wi-Fi or hotspot) for device syncing
  • At least 500 MB free disk space (depends on use)
  • Recommended: Temporarily disable VPN or firewall during installation if you encounter connectivity issues, then re-enable for security.

Installation Steps

1

Download the Installer

Click the Download Installer button at the top of this page (or use the direct link) and save the EXE to a folder you can access.

2

Run & Approve

Right-click → Run as administrator. If Windows SmartScreen appears, choose More info → Run anyway. Accept prompts and allow firewall access when asked.

3

Initial Setup

Choose a folder to use as the sync/store location, enable auto-start if desired, and click Finish. Open AkServer and follow on-screen instructions to connect devices.

How to Set Up AkServer

1

Bypass SmartScreen

Click "More Info" → "Run Anyway" to start setup.

2

Allow Installer

Windows may ask for permission. Click “Yes”.

3

Enable Auto Start

Let AkServer run on boot — ideal for backup PCs.

4

Allow in Firewall

Give permission when asked — this enables local access.

5

First Setup

After firewall approval, AkServer window opens — ready for action.

6

Connect & Scan

Click "Connect" and scan the QR or access the link directly.

7

Generate OTP

Click “Gen” to get a one-time code for secure device registration.

8

Setup Folder

Select a folder where files from your devices will sync.

9

Connect More Devices

Register and link multiple devices — all visible in one place.

10

Need More Help?

Click “More Info” on main screen to explore advanced options.

How to Set Up Web App

1

Open the Web App

Scan the QR code displayed on your AkServer host dashboard, or open the local Web App URL (e.g. https://192.168.x.x:5000) in your browser. Make sure the device you use is connected to the same local network as the AkServer host.

2

Browser Security Warning

Your browser may display "Your connection is not private" or a similar warning. This happens because AkServer serves a local certificate that can't be validated by public certificate authorities (the service runs entirely on your private network and does not use the public internet). This is expected for local-only servers.

How to proceed: click Advanced → then choose Proceed to <your-AkServer-URL> (unsafe). This is safe when you are on your local, trusted network and confirmed the link is from your AkServer host.

3

Login with OTP

After proceeding past the browser warning you'll reach the AkServer login page. Enter the One-Time Password (OTP) shown on the AkServer host (or generated by the host) and click Login. The OTP rotates for security — use the current code.

4

Upload Files

Open the Files tab. Use Upload or Select Folder to choose files from your device. The Web App shows upload progress; files are stored on the AkServer host and become immediately available to other connected devices on the same network.

5

View and Manage Files

Switch between Grid, List, or Details view to browse files. Click any file to preview or download it. Use the search bar to locate files quickly. Long-press (mobile) or right-click (desktop) for more file actions (rename, delete, share link if enabled).

6

Register Device & Settings

If prompted, give your device a friendly name so it’s recognizable in the AkServer host. Visit Settings to configure sync folders, auto-upload, and device permissions. Changes apply immediately across the local network.

7

Live Updates & Help

Check the Live Updates panel for announcements and quick tips. Visit the Help/FAQ section for troubleshooting steps (firewall rules, network checks). When finished, use Logout to terminate the session.

Troubleshooting & FAQ

Installer blocked by SmartScreen — what to do?

If Windows shows SmartScreen, click More info and then Run anyway. SmartScreen flags are common for new or locally distributed apps.

How do I verify the integrity of the installer?

Check the SHA256 checksum matches the value provided in the Download section. Use a tool like Windows 10 Hash Calculator.

Firewall prevents local access

When prompted, allow AkServer through your firewall. If needed, add a rule for the AkServer executable in Windows Defender Firewall > Allow an app or feature and enable private network access.

Devices cannot find the server

Ensure all devices are on the same Wi-Fi or hotspot. Disable VPNs or network filters temporarily. Check that AkServer shows "listening" in the main window.

What if AkServer doesn’t appear in the Start Menu?

Search for "AkServer" in the Windows search bar. If not found, navigate to C:\Program Files\AkServer and run AkServer.exe manually. Reinstall if the issue persists.

How to uninstall AkServer?

Go to Control Panel > Programs > Uninstall a program and select AkServer. Follow the uninstallation wizard.
